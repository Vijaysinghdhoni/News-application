# News Application
An Android News app built with Jetpack Compose and the latest android libraries.

# Tech Used
--**Jetpack Compose**
--**MVVM Clean Architecture**
--**Retrofit**
--**Room**
--**Dagger Hilt**
--**Material 3**
--**Kotlin flows and coroutines**
